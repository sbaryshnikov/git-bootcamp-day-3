# taskmanager

Учебный CLI-менеджер задач на Python. Позволяет добавлять задачи, просматривать список и помечать их как выполненные.

## Запуск

```bash
python3 -c "import tasks; tasks.add_task('купить хлеб'); tasks.list_tasks()"
```

## Установка

```bash
# Клонируйте репозиторий
git clone https://github.com

# Перейдите в папку проекта
cd taskmanager

# Проверьте работоспособность
python3 tasks.py
```

## API функции

<details>
<summary><b>Показать минимальный API в <code>tasks.py</code></b></summary>

Внутри модуля доступны следующие функции для интеграции:
* `add_task(title)` — добавляет новую задачу в список.
* `list_tasks()` — выводит все текущие задачи в консоль.
</details>
